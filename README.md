# Shadow_Group-Project

Judul Proyek: Shadow YouTube Analyzer

Deskripsi Singkat: 
Project ini bertujuan untuk Menganalisis performa video YouTube menggunakan Google Sheet untuk melihat jumlah views,likes, dan komentar secara otomatis, serta menampilkan ringkasan singkat video tersebut.

Petunjuk Penggunaan Aplikasi:
1. Input link youtube yang dicopy dari search bar, lalu paste pada text boxnya
   (Note: Jika ingin menginput lebih dari 1 link klik enter terlebih dahulu sebagai pemisah link 1 dengan lainnya)
2. Klik button "Mulai Analisis"
3. Setelah selesai maka akan muncul output yang menampilkan judul, URL youtube, ringkasan singkat, sentimen(positive, neutral, negative), jumlah likes, views, dan comments.

Petunjuk instalasi dan cara menjalankan proyek di lingkungan lokal:
1. Pada terminal setiap folder Back-end dan Front-end jalankan perintah "npm install"
2. Pada terminal folder Back-end jalankan perintah "npm run dev"
3. Pada terminal folder Front-end jalankan perintah "npm start", lalu otomatis akan terbuka pada local host perangkat kita.
